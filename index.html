Sobre mi

En 2019 me gradué como Ingeniero Agrimensor en la Universidad Nacional de La Plata, y tras dos años de ejercer la profesión me di cuenta que mi vocación estaba en otro lado. Así fue como en 2021 tome la decisión de estudiar "Analista Programador" en la Facultad de Informática y di mis primeros pasos en la programación.

En 2022, con los fundamentos que obtuve del primer año de facultad comencé a profundizar en el desarrollo web y opte por el camino del backend. Realice cursos y bootcamps de desarrollo de API REST con Java y Spring.

Actualmente estoy en cursando 2do año de facultad y estudiando por mi cuenta el Framework Angular para poder consumir los endpoint de mis APIs con mis propios desarrollos frontend, y poder desarrollar aplicaciones de forma integral siguiendo el patron Modelo Vista Controlador.

Proyectos

API REST - "SMS Weather"

Esta API implementa una base de datos MongoDB donde se almacenan numeros de telefono y ubicaciones, que son usadas para realizar consultas a la APIs "Geocoding API" y "Weather API" para obtener las coordenadas y el clima de una ciudad. Luego estos datos serian enviados a cada usuario via SMS, a traves de la integracion de Twilio SMS API. Ademas se implementa un sistema de verificacion de usuario con generacion de codigo aleatorio.

API REST - "ONG Somos Mas"

Proyecto grupal en la que participe. Puse a prueba mis conocimientos de Git para poder trabajar con otros 7 desarrolladores y un TL. Tambien experimente de las distintas ceremonias SCRUM. El proyecto consistio en desarrollar el backend para una ONG, con autenticacion con roles, integracion de "Amazom S3" y "Twilio SendGrid", y se le hizo testing con JUnit5 y Mockito
API REST - "Gestor de Empleados"

API que permite realizar operaciones CRUD sencillas sobre una base de datos MySQL. Este fue mi primer proyecto en Spring Boot, por lo que tiene excesivas lineas de comentarios y algunos errores (como no haber implementado una capa de servicios) pero nunca fue mejorado porque me es util para recordar como empece y lo que fui aprendiendo con el tiempo
